## Base URL

Todas as chamadas de API devem ser feitas para o endpoint da plataforma:

```
https://SEU-PROJETO.convex.site
```

<Note>
  Substitua `SEU-PROJETO` pela URL do Convex Site do SMS Infinity. Você pode encontrar essa URL na seção **API** do seu painel.
</Note>

## Formato das Respostas

Todas as respostas seguem o mesmo envelope JSON:

<CodeGroup>
```json Sucesso
{
  "ok": true,
  "data": { ... }
}
```

```json Erro
{
  "ok": false,
  "error": {
    "code": "INSUFFICIENT_BALANCE",
    "message": "Saldo insuficiente. Seu saldo atual é R$ 0,50 mas o número custa R$ 1,50."
  }
}
```
</CodeGroup>

## Acesso à API

O acesso à API é **restrito e controlado**. Para utilizá-la:

1. Entre em contato com nossa equipe solicitando acesso.
2. Após aprovação, acesse o **Painel → seção API** e gere seu token.
3. O token é exibido **uma única vez** — guarde-o em local seguro.

<Warning>
  A API debita seu saldo interno do SMS Infinity a cada compra. Certifique-se de ter créditos carregados antes de realizar ativações programáticas.
</Warning>
